https://s-link.herokuapp.com

Веб-сервис сокращения ссылок

Cервис задеплоин на вебхостинге heroku.

На главной странице форма ввода ссылки и кнопка сабмита. По сабмиту формы сервер 
отдает результат в виде сокращенной ссылки и выводит ее на экране.
Ввод ссылки, которую уже добавили генерирует новую кототкую ссылку.

Ссылки сохраняются в базу данных.
Есть авторизация и страница пользователя со статистикой.
Есть страница со статистикой всех сгенерированных ссылок и количесто переходов по каждой ссылке.
